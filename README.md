# sparrow-nfts

#### Start development

From a command line terminal :

1. Install dependencies
      ```cd sc && npm i```
2. Compile smart contracts
      ```cd sc && npx hardhat compile```
3. Test smart contracts
      ```cd sc && npx hardhat test```

#### Folder Structure
- sc: **Using Hardhat** to write, compile, debug smart contracts.
